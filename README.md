# 小序 (Xiao Xu)

一个集「打卡提醒」与「情感陪伴」于一体的跨平台桌面挂件（Widget），面向打工人日常，轻量、贴心、可爱。

## 亮点
- 透明无边框挂件，桌面常驻
- Hover 显示气泡话术，温柔小宠物机器人陪伴
- 签到 / 签退提醒 + 一键跳转打卡链接
- 久坐提醒（每 2 小时）
- 法定节假日感知（`timor.tech`）
- 天气感知（OpenWeather，恶劣天气自动提前提醒）
- 支持 OpenAI / Gemini / DeepSeek（可选 AI 话术）
- 置顶 / 鼠标穿透模式

## 使用方式
- **右键挂件**：打开设置、置顶、鼠标穿透、退出
- **托盘菜单**：显示/隐藏挂件、打开设置

## 设置面板
- 昵称
- 签到 / 签退时间
- 打卡链接
- 预计下班时间
- OpenWeather API Key + 经纬度
- AI 模式（OpenAI / Gemini / DeepSeek）

## 开发与运行
```bash
npm install
npm run dev
```

## 打包
```bash
npm run dist
```

## 自动构建与发布
- 推送到 `main`/`master` 会自动构建并上传产物
- 打 tag（例如 `v0.2.0`）会自动发布 Release 并附带安装包

## 备注
- macOS 首次打开可能提示“未知开发者”，可在「系统设置 → 隐私与安全性」选择继续打开
